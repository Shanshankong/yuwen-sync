💡 使用说明

1. 别人怎么用? 他打开应用 → 注册账号(默认 free) → 试 Day 1 + 任选 3 天
2. 别人付款后? 他把账号名发给你 → 你在这里搜账号 → 点【开通付费】 → 他下次同步立即解锁
3. 想清理测试账号? 点【删除】按钮,账号会被标记为已删除并从列表隐藏。
4. 删错了想恢复? 点右上角【显示已删除】查看,再点【恢复】
5. 备注怎么用? 记付款方式 / 金额 / 来源 / 优惠码,以后对账方便
6. 安全提醒: Worker URL 和管理员密码会暂存在本设备浏览器,公共电脑用完记得登出
